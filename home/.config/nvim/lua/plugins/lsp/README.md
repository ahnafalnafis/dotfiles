# LSP configuration

**Module structure:**

| Sub modules    | Description                            |
| -------------- | -------------------------------------- |
| init.lua       | Lazy spec root                         |
| config.lua     | LSP configuration                      |
| keymaps.lua    | LSP keymaps                            |
| completion.lua | Configuration for auto complete        |
| helpers.lua    | Helper functions                       |
| servers/       | Language server specific configuration |
